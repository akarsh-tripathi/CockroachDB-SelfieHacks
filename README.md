# CockroachDB-SelfieHacks

## Introduction

Using Cockroach DB , we made a bank system consiting of number of users and verifying account data using SQL connection strings


## Tech Stack

Our tech stack consists of the following:

* **Cockroach-DB** 
* **C#** 
* **.NET**
* **Postgress SQL**(npsql)
